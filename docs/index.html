Webpage Region Blur

Blur any part of a webpage with one bookmark. Useful for screenshots, screen sharing, tutorials, and hiding private information.

Drag me to your bookmarks bar

Drag the button above to your bookmarks bar. Then click it on any webpage.

Install in 3 steps

  1. 1 Show your browser bookmarks bar.
  2. 2 Drag the blue Region Blur button to the bookmarks bar.
  3. 3 Open any webpage and click the bookmark.

What you can do

Blur private data Cover names, emails, balances, addresses, messages, and IDs.
Move and resize Drag the blur box anywhere and resize it from the corners.
Multiple blur boxes Click the bookmark again to create another blur area.
No extension It is only a bookmark. No account, no permissions, no tracking.

Manual install

If drag-and-drop does not work, copy this code and paste it into the URL field of a new bookmark.

How to use

  1. 1 Click the bookmark to create a blur box.
  2. 2 Drag the box to move it.
  3. 3 Drag a corner to resize it.
  4. 4 Hold Alt and scroll to change blur strength.
